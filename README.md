# AI Music Generation using LSTM 🎵

This project generates AI-based piano music using LSTM neural networks and MIDI files.

## Features
- MIDI preprocessing
- LSTM deep learning model
- AI melody generation
- MIDI output generation

## Technologies Used
- Python
- TensorFlow / Keras
- music21
- NumPy

## Files
- preprocess.py
- model.py
- train.py
- generate.py

## Run Project

```bash
python preprocess.py
python train.py
python generate.py
```

## Output
The model generates AI-created MIDI music files.
